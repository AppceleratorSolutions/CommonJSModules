Ti.MapCluster Module

Author

Davide Cassenti

Use the module

1. Set the TITANIUM_SDK environment variable to the Titanium SDK to use 2. Run the build.py python script 3. Unzip the built module in the Titanium modules directory 4. Copy the example/pins directory in the app resources directory (those pins can be customized) 5. Use the module by requiring it with require('ti.mapcluster')
